    FOSS4G4 airborne

    Adam Steer, Norsk Polarinstitutt og OSGeo

    This talk will discuss FOSS4G technologies for collecting and managing airborne surveying data, with an emphasis on OpenDroneMap for data processing, and the PDAL/Entwine stack for point clouds. It will show a few “extreme use cases” for OpenDroneMap and a walkthrough of how ODM was modified to use Polar Stereographic coordinates where UTM might be less reliable. Using Norways Høydedata collection it will then walk through data exploitation with Entwine Point Tiles and COPC – cloud optimized point clouds, and explain which should be used when. In short, it will show how minimize storage overhead for multiple deliverable outputs.

  • Kl. 14.20

    Biologisk mangfold og åpen programvare - store systemer og store data

    Dag Endresen, Naturhistorisk museum

    GBIF - Global Biodiversity Information Facility - er et internasjonalt distribuert nettverk og infrastruktur for forskningsdata som er finansiert av medlemsland og som gir fri og åpen tilgang for alle til artsdata og informasjon om biologisk mangfold fra hele verden. Den norske GBIF-noden arbeider sammen med Living Norway Ecological Data Network for å etablere FAIR prinsippene for datahåndering innenfor økologisk forskning i Norge. Sammen med de store forskningsinfrastrukturer for biologiske data i Europa deltar vi også i et Horisont Europa prosjekt - Biodiversity Digital Twin (BioDT) - for å bygge digitale tvillinger for biologiske systemer som kan hjelpe oss å forstå hvordan disse vil respondere på de store endringer vi idag opplever for klima og biodiversitet. BioDT inngår i tunge Europeiske strategiske program for bevaring av biodiversitet og forvaltning av naturressurser. BioDT henter åpne og veldig store geo-data kilder inn til Europas hurtigste tungregnemaskin, LUMI, som ble driftsatt i Finland i juni 2022, for å bygge datamodeller for de digitale tvillingene.
